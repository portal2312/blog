Develop new apps on Docker

Docker에 새로운 앱을 개발하기 시작한 경우 Docker에서 가장 많은 혜택을 얻으려는 가장 일반적인 패턴을 이해하려면 이 리소스를 확인하십시오.

  • Dockerfile 에서 image 만드는 방법 배우기
  • 이미지를 설립 상태를 유지하려면 다단계 빌드 사용하기
  • 응용 프로그램 데이터를 사용중인 volumns 과 bind mounts 관리하기
  • 너의 app 을 swarm service 처럼 규모 갖추기
  • 너의 app stack 을 작성 파일을 사용하여 정의하기
  • 일반적인 애플리케이션 개발 모범 사례

Advanced development with the SDK or API

SDK 또는 API 를 통한 고급 개발

Dockerfiles를 작성하거나 파일을 작성하고 Docker CLI를 사용하여 응용 프로그램을 개발하는 데 익숙한 경우 Go 또는 Python 용 Docker Engine SDK를 사용하거나 HTTP API를 직접 사용하여 다음 단계로 가져갈 수 있습니다.

Reference

https://docs.docker.com/develop/
