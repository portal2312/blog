CSS 선택기는 스타일을 지정할 HTML 요소를 "찾기"(또는 선택)하는 데 사용됩니다.

CSS 선택기를 5 가지 범주로 나눌 수 있습니다:

  • 간단한 선택기 (이름, ID, 클래스를 기준으로 요소 선택)
  • Combinator(조합기) 선택기 (그들 간의 특정 관계에 따라 요소를 선택)
  • 의사-클래스 선택기 (특정 상태에 따라 요소 선택)
  • 의사-요소 선택기 (요소의 일부를 선택하고 스타일 지정)
  • 속성 선택기 (특성 또는 속성 값을 기반으로 요소 선택)
