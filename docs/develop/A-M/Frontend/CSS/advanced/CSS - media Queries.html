CSS2 Media Types 소개

CSS2 에 도입 된 @media 규칙은 다양한 미디어 유형에 대해 서로 다른 스타일 규칙을 정의할 수 있게했습니다.

예: 컴퓨터 화면, 프린터, 핸드 헬드 장치, TV 유형 장치 등의 스타일 규칙 집합이 하나있을 수 있습니다.

불행히도 이러한 용지 종류는 인쇄 용지 종류 이외의 장치에서 많은 지원을 받지 못했습니다.

CSS3 Media Queries 소개

CSS3의 미디어 쿼리는 CSS2 미디어 유형 아이디어를 확장했습니다.

장치 유형을 찾는 대신 장치의 기능을 살펴 봅니다.

미디어 쿼리를 사용하여 다음과 같은 많은 것을 확인할 수 있습니다.

  • 뷰포트의 너비와 높이
  • 장치의 너비와 높이
  • 방향 (태블릿/전화가 가로 또는 세로 모드에 있습니까?)
  • 해결

미디어 쿼리 사용은 데스크탑, 랩톱, 태블릿 및 휴대폰 (예 : iPhone 및 Android 전화)에 맞춤형 스타일 시트를 제공하는 데 널리 사용되는 기술입니다.

Browser Support

Property Chome IE FF SF OP
@media 21.0 9.0 3.5 4.0 9.0

Syntax

media query 는 미디어 유형으로 구성되며 true 또는 false로 해석되는 하나 이상의 표현식을 포함 할 수 있습니다.

@media not|only mediatype and (expressions) {
  CSS-Code;
}

지정된 매체 유형이 문서가 표시되는 장치 유형과 일치하고 매체 조회의 모든 표현식이 참인 경우 조회 결과는 참입니다. 미디어 쿼리가 true이면 일반적인 계단식 규칙에 따라 해당 스타일 시트 또는 스타일 규칙이 적용됩니다.

not 또는 only 연산자를 사용하지 않으면 매체 유형은 선택 사항이며 all 유형이 암시됩니다.

매체마다 다른 스타일 시트를 가질 수도 있습니다:

<link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">

CSS3 Media Types

value description
all 모든 매체 유형 장치에 사용
print 프린터 에서 사용
screen 컴퓨터 스크린, 태블릿, 스마트 폰 에서 사용
speech 스크린 리더 에서 사용

Examples

body {
  background-color: pink;
}

@media screen and (min-width: 480px) {
  body {
    background-color: lightgreen;
  }
}

CSS @media Reference

모든 미디어 유형 및 기능 / 표현에 대한 전체 개요는 CSS 참조의 @media 규칙을 참조하십시오.

References

https://www.w3schools.com/css/css3_mediaqueries.asp
