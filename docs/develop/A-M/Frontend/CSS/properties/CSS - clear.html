<p class="clear">This is also some text. This is also some text. This is also some text. This is also some text. This is also some text. This is also some text.</p>
<p><strong>Remove the "clear" class to see the effect.</strong></p>

Definition and Usage

정의와 사용법.

clear property 는 float element 가 floating 할 수 없는 요소의 측면을 지정합니다.

Values

No floating elements allowed on:

Value Description
none Default. Allows floating elements on both sides
left left side
right right side
both left or the right side
initial 이 속성을 기본 값으로 합니다. Read about initial
inherit 부모 element 부터 이 property 를 상속 받습니다. Read about inherit
