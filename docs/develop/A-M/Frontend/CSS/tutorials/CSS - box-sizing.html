CSS - box-sizing

Table of contents

  • Without the box-sizing Property
  • With the box-sizing Property
  • References

element 의 전체 width와 height에 padding과 border를 포함시킬 수 있습니다.

Without the box-sizing Property

  • actual width of an element = width + padding + border
  • actual height of an element = height + padding + border

의미: 요소의 너비 / 높이를 설정하면 요소의 경계와 패딩이 요소의 지정된 너비 / 높이에 추가되므로 요소가 설정 한 것보다 더 크게 나타납니다:

.div1 {
  width: 300px;
  height: 100px;
  border: 1px solid blue; 
}

.div2 {
  width: 300px;
  height: 100px;
  padding: 50px;
  border: 1px solid red;
}
  • div width = 300 + (1 x2)
  • div2 width = 300 + (1 x2) + (50 x2)

With the box-sizing Property

element 의 전체 너비와 높이에 패딩과 테두리를 포함시킬 수 있습니다.

box-sizing: border-box; element padding 및 border에 width와 height가 포함됩니다:

.div1 {
  width: 300px;
  height: 100px;
  border: 1px solid blue;
  box-sizing: border-box;
}

.div2 {
  width: 300px;
  height: 100px;
  padding: 50px;
  border: 1px solid red;
  box-sizing: border-box;
}
  • div1 width = 300
  • div2 width = 300

box-sizing: border-box;을 사용한 결과가 훨씬 더 좋습니다. 많은 개발자가 자신의 페이지에있는 모든 요소가이 방식으로 작동하기를 원합니다.

아래 코드는 모든 elements 의 크기가 이보다 직관적으로 유지되도록합니다. 많은 브라우저들이 이미 많은 elements 양식에 경우 box-sizing: border-box;를 사용합니다. (모두는 아니지만 input과 text area 의 witdh: 100%)

이것을 모든 요소에 적용하는 것이 안전하고 현명합니다:

* {
  box-sizing: border-box;
}
