position 속성은 요소에 사용되는 위치 결정 방법의 다음과 같은 유형을 지정합니다:

Property

  • static (정적)
  • relative (상대적)
  • fixed (고정)
  • absolute (절대)
  • sticky (달라 붙는, 스티커)

그런 다음 속성 top, bottom, left 및 right 의 값에 의해 배치됩니다.
그러나 position 속성을 먼저 설정하지 않으면 이러한 속성이 작동하지 않습니다.
위치 값에 따라 다르게 작동합니다.

position: static

HTML 요소는 기본적으로 정적 위치에 있습니다.

정적 위치 요소는 top, bottom, left 및 right 속성의 영향을 받지 않습니다.

항상 페이지의 정상적인 흐름에 따라 배치됩니다.

div.static {
  position: static;
  border: 3px solid #73AD21;
}

position: relative

정상 위치를 기준으로 배치됩니다.

상대적으로 배치 된 요소의 위쪽, 오른쪽, 아래쪽 및 왼쪽 속성을 설정하면 일반 위치에서 멀어 지도록 조정됩니다.

다른 내용은 요소가 남긴 간격에 맞게 조정되지 않습니다.

div.relative {
  position: relative;
  left: 30px;
  border: 3px solid #73AD21;
}

position: fixed

뷰포트를 기준으로 배치되므로 페이지가 스크롤 되더라도 항상 같은 위치에 유지됩니다.

top, right, bottom 및 left 속성은 요소를 배치하는 데 사용됩니다.

고정 된 요소는 일반적으로 위치했던 페이지에 공백을 남기지 않습니다.

페이지 오른쪽 하단에 고정 요소가 있습니다.

div.fixed {
  position: fixed;
  bottom: 0;
  right: 0;
  width: 300px;
  border: 3px solid #73AD21;
}

position: absolute

(고정 된 것처럼 뷰포트를 기준으로 배치하는 대신) 가장 가까운 배치 된 조상을 기준으로 배치됩니다.

단, 절대 위치 요소에 위치된 조상이 없으면 문서 본문을 사용하고 페이지 스크롤과 함께 이동합니다.

Note: "positioned" 요소는 static외의 위치에 있는 입니다.

div.relative {
  position: relative;
  width: 400px;
  height: 200px;
  border: 3px solid #73AD21;
} 

div.absolute {
  position: absolute;
  top: 80px;
  right: 0;
  width: 200px;
  height: 100px;
  border: 3px solid #73AD21;
}

Note: 상위 element 안에 하위 element 로 정의시
정렬(default: left) 기준으로 상위 element border 에서부터 시작하여 표현됩니다.

position: sticky

사용자의 스크롤 위치를 기준으로 배치됩니다.

스티커 요소는 스크롤 위치에 따라 relative 및 fixed 사이를 전환합니다.

주어진 오프셋 위치가 뷰포트에서 충족 될 때까지 상대적으로 배치 된 다음 위치에 고정됩니다(예: position: fixed).

Note: Internet Explorer, Edge 15 및 이전 버전은 고정 위치를 지원하지 않습니다.
Safari에는 -webkit- 접두사가 필요합니다 (아래 예 참조).
또한 고정 위치가 작동하려면 top, right, bottom 또는 left 속성중 하나 이상을 지정해야 합니다.

스크롤 위치에 도달하면 고정 요소가 페이지 상단(top: 0)에 고정됩니다.

div.sticky {
  position: -webkit-sticky; /* Safari */
  position: sticky;
  top: 0;
  background-color: green;
  border: 2px solid #4CAF50;
}

Overlapping Elements

요소가 배치되면 다른 요소와 겹칠 수 있습니다.

z-index 속성은 요소의 스택 순서를 지정합니다 (요소는 다른 요소의 앞 또는 뒤에 배치되어야 함).

요소는 양수 또는 음수 스택 순서를 가질 수 있습니다.

img {
  position: absolute;
  left: 0px;
  top: 0px;
  z-index: -1;
}

Positioning Text in an Image

.center {
  position: absolute;
  left: 0;
  top: 50%;
  width: 100%;
  text-align: center;
  font-size: 18px;
  /* background-color: red; */
}
