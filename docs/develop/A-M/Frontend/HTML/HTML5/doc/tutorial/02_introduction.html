What is HTML?

HTML은 웹 페이지를 만들기 위한 표준 마크업 언어입니다.

  • HTML은 하이퍼 텍스트 마크 업 언어
  • HTML은 마크 업을 사용하여 웹 페이지의 구조를 설명합니다.
  • HTML 요소는 HTML 페이지의 구성 요소입니다.
  • HTML 요소는 태그로 표현됩니다.
  • HTML 태그는 "heading", "paragraph", "table"등과 같은 내용의 레이블을 표시합니다.
  • 브라우저는 HTML 태그를 표시하지 않지만 이를 사용하여 페이지의 내용을 렌더링합니다.

A Simple HTML Document

Example

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Example Explained

  • The <!DOCTYPE html> declaration defines this document to be HTML5

이 문서는 HTML5 이다 를 정의.

  • The <html> element is the root element of an HTML page

HTML 페이지 의 root 요소.

  • The <head> element contains meta information about the document

이 문서에 대한 meta 정보를 포함.

  • The <title> element specifies a title for the document

이 문서의 제목을 명세.

  • The <body> element contains the visible page content

보이는 페이지 content를 포함.

  • The <h1> element defines a large heading

큰 heading(제목)을 정의.

  • The <p> element defines a paragraph

단락을 정의.

The <!DOCTYPE> Declaration

<!DOCTYPE> 선언은 문서 유형을 나타내며 브라우저가 웹 페이지를 올바르게 표시하도록 합니다.

HTML 태그보다 먼저 페이지 상단에 한 번만 나타나야합니다.

<!DOCTYPE> 선언은 대소 문자를 구별하지 않습니다.
