HTML Headings

<h1> 은 가장 중요한 제목을 정의 함.

중요도: <h1> > <h2> > ... > <h6> 까지 있음.

Examples:

<h1>AAA</h1>
<h2>AAA</h2>
<h3>AAA</h3>

Paragraphs

단락을 정의.

<p>AAA</p>
<p>BBB</p>

Links

링크를 정의.

<a href="https://www.google.com">go to google</a>

Images

이미지 정의.

다양한 속성들(src, alt, width, height)

<img src="" alt="" width="" height="">
