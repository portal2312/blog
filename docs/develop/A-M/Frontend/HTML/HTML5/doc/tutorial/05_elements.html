HTML Elements

시작 태그와 종료 태그로 구성.

내용이 없는 HTML 요소를 empty elements 라고 합니다.

empty elements 는 <br> 요소 (줄 바꿈을 나타냄)와 같은 끝 태그가 없습니다.

Nested HTML Elements

HTML 요소는 중첩 될 수 있습니다 (요소에는 요소가 포함될 수 있음).

Explained

<html> </html> 전체 문서 정의.

<body> </body> 요소는 문서 본문을 정의.

<h1> </h1> 제목을 정의.

<p> </p> 단락(절)을 정의.

Forget the End Tag

위의 예제는 모든 브라우저에서 작동합니다. 닫기 태그는 선택 사항으로 간주되기 때문입니다.

이것을 절대 의지하지 마십시오.

종료 태그를 잊어 버리면 예기치 않은 결과 및 / 또는 오류가 발생할 수 있습니다.

Empty HTML Elements

내용이 없는 HTML 요소를 empty elements 라고 합니다.

<br> 은 닫는 태그가 없는 empty elements 입니다 (
태그는 줄 바꿈을 정의합니다).

empty elements 는 다음과 같이 시작 태그에서 "닫을"수 있습니다.

<br /> <br /> HTML5에서는 빈 요소를 닫을 필요가 없습니다.

그러나보다 엄격한 유효성 검사를 원하거나 XML 파서로 문서를 읽을 수 있도록 해야하는 경우 모든 HTML 요소를 올바르게 닫아야합니다.

Use Lowercase Tags

HTML 태그는 대소 문자를 구분하지 않습니다. <P>는 <p>와 같습니다.

HTML5 표준은 소문자 태그를 요구하지 않지만 W3C는 HTML에서 소문자를 권장하고

XHTML과 같은 더 엄격한 문서 유형을 위해서는 소문자를 요구합니다.
