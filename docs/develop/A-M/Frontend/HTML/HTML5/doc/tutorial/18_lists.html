Unordered HTML List

태그: <ul> <li>...</li>... </ul>

list 는 <li>

목록 항목은 기본적으로 글 머리 기호 (작은 검은 색 원)로 표시됩니다:

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Unordered HTML List - Choose List Item Marker

CSS list-style-type 속성은 목록 항목 마커의 스타일을 정의하는 데 사용됩니다:

Value Discription
disc bullet
circle circle
square square
none not be marked 
<ul style="list-style-type:disc">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Ordered HTML List

태그: <ol> <li>...</li>... </ol>

list 는 <li>

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Ordered HTML List - The Type Attribute

<ol> 태그의 type 속성은 목록 항목 마커의 유형을 정의합니다:

Type Description
type="1" numbers
type="A" uppercase
type="a" lowercase
type="I" uppercase roman numbers
type="i" lowercase roman numbers 
<ol type="1">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

HTML Description Lists

HTML은 설명 목록도 지원합니다.

discription list 는 용어와 함께 각 용어의 설명의 목록이다.

<dl> tag 는 내용 list 를 정의한다.

<dt> tag 는 용어(이름, term)을 정의한다.

<dd> 각 이름의 기술(describes)한다.

<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>

Nested HTML Lists

목록은 중첩 될 수 있습니다 (목록 내부 목록):

<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

Note: 목록 항목에는 새 목록과 이미지 및 링크와 같은 기타 HTML 요소가 포함될 수 있습니다.

Horizontal Lists

HTML 목록은 CSS를 사용하여 다양한 방식으로 스타일을 지정할 수 있습니다.

한 가지 인기있는 방법은 목록을 가로로 스타일 지정하여 메뉴를 만드는 것입니다.

<!DOCTYPE html>
<html>
<head>
<style>
ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
    overflow: hidden;
    background-color: #333333;
}

li {
    float: left;
}

li a {
    display: block;
    color: white;
    text-align: center;
    padding: 16px;
    text-decoration: none;
}

li a:hover {
    background-color: #111111;
}
</style>
</head>
<body>

<ul>
  <li><a href="#home">Home</a></li>
  <li><a href="#news">News</a></li>
  <li><a href="#contact">Contact</a></li>
  <li><a href="#about">About</a></li>
</ul>

</body>
</html>

Chapter Summary

  • Use the HTML <ul> element to define an unordered list
  • Use the CSS list-style-type property to define the list item marker
  • Use the HTML <ol> element to define an ordered list
  • Use the HTML type attribute to define the numbering type
  • Use the HTML <li> element to define a list item
  • Use the HTML <dl> element to define a description list
  • Use the HTML <dt> element to define the description term
  • Use the HTML <dd> element to describe the term in a description list
  • Lists can be nested inside lists
  • List items can contain other HTML elements (목록 항목에는 다른 HTML 요소가 포함 가능)
  • Use the CSS property float:left or display:inline to display a list horizontally

Reference

https://www.w3schools.com/html/html_lists.asp
