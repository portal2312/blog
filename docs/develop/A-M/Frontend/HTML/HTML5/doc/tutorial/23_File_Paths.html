HTML File Paths

파일 경로는 웹 사이트의 폴더 구조에있는 파일의 위치를 ​​설명합니다.

파일 경로는 다음과 같은 외부 파일에 연결할 때 사용됩니다.

  • Web pages
  • Images
  • Style sheets
  • JavaScripts

Absolute File Paths

절대 파일 경로

인터넷 파일의 전체 URL입니다.

<img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">

Relative File Paths

상태 파일 경로

현재 페이지를 기준으로 파일을 가리킵니다.

이 예제에서 파일 경로는 현재 웹의 루트에있는 images 폴더의 파일을 가리킵니다.

<img src="/images/picture.jpg" alt="Mountain">

이 예제에서 파일 경로는 현재 폴더에있는 images 폴더의 파일을 가리킵니다:

<img src="images/picture.jpg" alt="Mountain">

이 예제에서 파일 경로는 현재 폴더의 한 수준 위의 폴더에있는 images 폴더의 파일을 가리킵니다:

<img src="../images/picture.jpg" alt="Mountain">

Best Practice

가능한 경우 상대 파일 경로를 사용하는 것이 좋습니다.

상대 파일 경로를 사용할 때 웹 페이지는 현재 기본 URL에 바인딩되지 않습니다. 모든 링크는 자신의 컴퓨터 (localhost)뿐만 아니라 현재 공개 도메인 및 향후 공개 도메인에서 작동합니다.

Reference

https://www.w3schools.com/html/html_filepaths.asp
