HTML <kbd> For Keyboard Input

HTML <kbd> 요소는 키보드 입력 또는 음성 명령과 같은 사용자 입력을 나타냅니다.

<kbd> 태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니다.

Example:

<p>Save the document by pressing <kbd>Ctrl + S</kbd></p>

Result:

Save the document by pressing Ctrl + S

HTML <samp> For Program Output

HTML <samp> 요소는 프로그램 또는 컴퓨팅 시스템의 출력을 나타냅니다.

<samp> 태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니다.

Example:

<p>If you input wrong value, the program will return <samp>Error!</samp></p>

Result:

If you input wrong value, the program will return Error!

HTML <code> For Computer Code

HTML <code> 요소는 컴퓨터 코드 조각을 정의합니다.

<code> 태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니다.

Example:

<code>
x = 5;
y = 6;
z = x + y;
</code>

Result:

x = 5; y = 6; z = x + y;

<code> 요소는 여분의 공백과 줄 바꿈을 보존하지 않습니다.

이 문제를 해결하려면 <code> 요소를 <pre> 요소 안에 넣을 수 있습니다.

Example:

<pre>
<code>
x = 5;
y = 6;
z = x + y;
</code>
</pre>

Result: 


x = 5;
y = 6;
z = x + y;

HTML <var> For Variables

HTML 요소는 변수를 정의합니다.

변수는 수학 표현식의 변수이거나 프로그래밍 컨텍스트의 변수 일 수 있습니다.

Example:

Einstein wrote: <var>E</var> = <var>mc</var><sup>2</sup>.

Result:

Einstein wrote: E = mc2.

Reference

https://www.w3schools.com/html/html_computercode_elements.asp
