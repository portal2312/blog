* javascript 내장함수

> a = function() {};
> b = 1000;  //b = b/1000 초이다

------------------------------------------------
1. setInterval();   // = 시간마다 실행됨

> setInterval( a, b );  // 시작
> clearInterval();      // 중지

------------------------------------------------
2. setTimeout();    // = 한번만 실행됨

> setTimeout( a, b );  // 시작
> clearTimeout();      // 중지
------------------------------------------------
