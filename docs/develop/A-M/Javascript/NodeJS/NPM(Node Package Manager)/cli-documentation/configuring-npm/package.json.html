private

"private": true 를 package.json에 설정하면 npm은 게시를 거부합니다.

개인 저장소가 실수로 게시되는 것을 방지하는 방법입니다.
지정된 패키지가 특정 레지스트리 (예: 내부 레지스트리)에만 공개되도록 하려면 publishConfig 아래 설명된 사전을 사용하여 registry 공개시 구성 매개 변수를 대체하십시오.

References

https://docs.npmjs.com/files/package.json
