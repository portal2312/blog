Installation

ESLint

sudo npm install -g eslint
# ...

version 확인하기:

sudo npm ls -g | grep eslint

├─┬ eslint@5.3.0
│ ├─┬ eslint-scope@4.0.3
│ ├── eslint-utils@1.3.1
│ ├── eslint-visitor-keys@1.0.0
├─┬ eslint-config-airbnb@17.1.0
│ ├─┬ eslint-config-airbnb-base@13.1.0
│ │ └── eslint-restricted-globals@0.1.1
├─┬ eslint-config-prettier@4.2.0
├─┬ eslint-plugin-import@2.17.2
│ ├── eslint-import-resolver-node@0.3.2
│ ├─┬ eslint-module-utils@2.4.0
├─┬ eslint-plugin-jsx-a11y@6.2.1
├─┬ eslint-plugin-react@7.12.4

npx 명령어를 이용하여 eslint-config-airbnb install:

sudo npx install-peerdeps -g eslint-config-airbnb

# ...
SUCCESS eslint-config-airbnb    
  and its peerDeps were installed successfully.

Init

ESLint

npm install --global 로 설치했기 때문에 별도 eslint init 하지 않는다.

앞으로 관리할 프로젝트 경로로 이동후 .eslintrc 작성하기:

{
  "env": {
    "browser": true
  },
  "extends": [
    "airbnb",
    "plugin:@sencha/extjs/recommended",
    "plugin:prettier/recommended"
  ],
  "globals": {
    "zenlog": false,
    "echarts": false,
    "$": false,
    "_": false,
    "__": false,
    "moment": false,
    "ReconnectingWebSocket": false,
    "domtoimage": false
  },
  "plugins": [
    "@sencha/extjs",
    "prettier"
  ],
  "rules": {
    "object-shorthand": ["error", "properties"],
    "func-names": ["error", "as-needed"],
    "no-param-reassign": ["error", { "props": false }],
    "@sencha/extjs/no-private-method-override": "off"
  }
}

Prettier

앞으로 관리할 프로젝트 경로로 이동후 .prettierrc 작성하기:

{
  "printWidth": 80,
  "singleQuote": true
}
