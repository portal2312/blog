특성 교차 - 비선형성 인코딩

Table of contents

  • 특성 교차의 종류
  • Reference

비선형 문제를 해결하려면 특성 교차를 만드세요. 특성 교차는 두 개 이상의 입력 특성을 곱하여 특성 공간에서 비선형성을 인코딩하는 합성 특성입니다. 교차라는 용어는 교차 곱에서 따온 것입니다. x1 과 x2 를 교차하여 x3 이라는 특성 교차를 만들어 보겠습니다.

x3 = x1 * x2

새로 만든 특성 교차는 다른 특성과 마찬가지로 취급합니다. 선형 수식은 다음과 같습니다.

y = b + w1x1 + w2x2 + w3x3

선형 알고리즘에서 w1 및 w2 와 마찬가지로 w3 의 가중치를 학습 가능.
즉, 이 비선형 정보를 인코딩해도 w3 의 값을 결정하기 위해 선형 모델의 학습 방식을 변경하지 않아도 됨.

특성 교차의 종류

여러 종류의 특성 교차를 만들 수 있다. 예:

  • [A X B]: 두 특성의 값을 곱하여 구성되는 특성 교차
  • [A x B x C x D x E]: 다섯 개의 특성 값을 곱하여 구성되는 특성 교차
  • [A x A]: 단일 특성을 제곱하여 구성되는 특성 교차

확률적 경사하강법을 활용하여 선형 모델을 효율적으로 학습 가능.
결과적으로 예전부터 조정된 선형 모델을 특성 교차로 보완하는 방법으로 모델을 대규모 데이터 세트에 효율적으로 학습시켜 왔습니다.
