학습 목표

  • 실제의 ML 모델에서 결함을 파악합니다.

요약

효과적인 ML 가이드라인

  • 최초 모델은 단순하게 유지, 구성하기.
  • 데이터 파이프라인의 정확성을 확보하는 데 중점을 둡니다.
  • 학습과 평과를 위해 간단하고 관찰 가능한 측정항목을 사용합니다.
  • 입력 특성을 관리하고 관찰합니다.
  • 모델 구성을 코드로 취급하여 검토하고 체크인합니다.
  • 모든 실험 결과를 기록하되, 특히 '실패한 결과'를 기록합니다.

기타 리소스
