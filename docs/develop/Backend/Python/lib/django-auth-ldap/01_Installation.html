Installation

Table of contents

$ pip install django-auth-ldap

python-ldap> = 3.0이 필요합니다.

시스템에서 OpenLDAP 라이브러리와 헤더를 사용할 수 있어야합니다.

$ pip install python-ldap
$ sudo apt install slapd ldap-utils

Django 프로젝트에서 auth 백엔드를 사용하려면 AUTHENTICATION_BACKENDS에 'django_auth_ldap.backend.LDAPBackend'를 추가하십시오. INSTALLED_APPS에 아무 것도 추가하지 마십시오.

settings.py:

AUTHENTICATION_BACKENDS = [
    'django_auth_ldap.backend.LDAPBackend',
]

LDAPBackend는 사용자 지정 사용자 모델에서 작동해야하지만 데이터베이스가 있다고 가정합니다.

Note

LDAPBackend는 ModelBackend에서 상속하지 않습니다. LDAP 서버에서 그룹 구성원을 그리도록 LDAPBackend를 독점적으로 구성하여 LDAPBackend를 독점적으로 사용할 수 있습니다. 그러나 개별 사용자에게 권한을 할당하거나 Django 내 그룹에 사용자를 추가하려면 두 백엔드를 모두 설치해야합니다. py AUTHENTICATION_BACKENDS = [ 'django_auth_ldap.backend.LDAPBackend', 'django.contrib.auth.backends.ModelBackend', ]
