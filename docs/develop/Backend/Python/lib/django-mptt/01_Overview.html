Overview

What is Modified Preorder Tree Traversal?

MPTT는 데이터베이스에 계층적 데이터를 저장하는 기술입니다.

목표는 검색 작업을 매우 효율적으로 만드는 것입니다.

이 효율성에 대한 절충은 트리 구조를 항상 양호한 상태로 유지하는 데 필요한 추가 작업이 있기 때문에 트리 주변에서 삽입 및 항목 이동을 수행하는 것이 더 복잡하다는 것입니다.

What is django-mptt?

django-mptt는 자신의 장고 모델로 MPTT를 사용하기 쉽게 만드는 재사용 가능한 장고 응용 프로그램입니다.

데이터베이스 테이블을 트리 구조로 관리하는 세부 사항을 처리하고 모델 인스턴스의 트리 작업을위한 도구를 제공합니다.

Feature overview

  • 모델의 간단한 등록 - 트리 구조에 필요한 필드가 자동으로 추가됩니다.
  • 트리 구조는 모델 인스턴스를 작성 또는 삭제하거나 인스턴스의 상위를 변경할 때 자동으로 갱신됩니다.
  • 트리의 각 레벨은 선택한 필드(들)로 자동 정렬됩니다.
  • 등록된 각 모델에 새로운 모델 메소드가 추가되었습니다.
    • Tree 에서 위치 변경
    • 조상, 형제, 자손 검색
    • 자손 계산
    • 다른 트리 관련 작업
  • TreeManager 매니저가 등록된 모든 모델에 추가됩니다. 이것은 다음과 같은 방법을 제공합니다:
    • Tree 주변의 노드 이동 또는 다른 트리로 이동
    • Tree 의 아무곳에나 노드를 삽입하십시오.
    • Tree 에 대한 MPTT 필드를 다시 빌드하십시오 (django 외부에서 대량 업데이트를 할 때 유용합니다)
  • 트리 모델의 양식 필드 트리 모델의 유틸리티 함수.
  • 트리를 렌더링하기위한 템플릿 태그 및 필터.
