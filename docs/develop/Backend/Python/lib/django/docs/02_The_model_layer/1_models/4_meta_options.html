unique_together

Options.unique_together

함께 취한 필드 이름 집합은 고유해야합니다.

unique_together = (("driver", "restaurant"),)

이것은 함께 고려할 때 고유해야하는 튜플의 튜플입니다.

Django 관리자에서 사용되며 데이터베이스 수준에서 적용됩니다.

(즉, 적절한 UNIQUE 문이 CREATE TABLE 문에 포함됨).

편의상 unique_together는 단일 필드 집합을 처리 할 때 단일 튜플이 될 수 있습니다.

unique_together = ("driver", "restaurant")

ManyToManyField는 unique_together에 포함될 수 없습니다.(이것이 의미하는 바는 분명하지 않음)

ManyToManyField와 관련된 고유성의 유효성을 검사해야하는 경우 신호 또는 명시적 모델을 사용해보십시오.

제약 조건을 위반했을 때 모델 유효성 검사 중에 제기된 ValidationError는

unique_together 오류 코드를 가집니다.
