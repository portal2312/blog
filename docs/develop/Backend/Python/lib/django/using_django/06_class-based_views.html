Decorating class-based views

클래스 기반 뷰의 확장은 믹스 인을 사용하는 것에 국한되지 않습니다. 데코레이터를 사용할 수도 있습니다. 클래스 기반 뷰는 함수가 아니기 때문에, 이들을 꾸미는 것은 as_view ()를 사용하는지 또는 서브 클래스를 생성하는지에 따라 다르게 작동합니다.
