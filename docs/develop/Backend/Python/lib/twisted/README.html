Installation

  • python 실행으로 설치 하는경우 python2 인지 python3 인지 주의할 것.

  • pip 설치시 pip2, pip3, pip(나중에 설치된 pip로 인지함) 인지 주의 할 것.

Python3.x Install Twisted, PIP Package Not Found (for Linux)

  • 경로 확인

    • 2.7.x

      • /usr/local/lib/python2.7 = 설치된 파일 경로
      • /usr/local/lib/python2.7/site-packages = python package 없음
      • /usr/lib/python2.7/site-packages = 모두 여기 있음

    • 3.x

      • /usr/lib/python3.4 = Package Manager 설치시 경로
      • /usr/lib/python3.4/site-packages = python package 없음
      • /usr/local/lib/python3.5 = Compile 기본 설치시 경로
      • /usr/local/lib/python3.5/site-packages = 모두 여기 있음

  • pip3 설치 필요 (pip != pip3), python-pip 문서 참조

  • Python3 설치시 자동 설치 됨

  • compile 로 python3 설치할 경우(package 설치 없이) 발생

    • pip install twisted (for Winodws)

import twisted error

  • pip install twisted 이후 /usr/lib/python2.7/site-packages/ 안에 설치되서 /usr/loca/bin/python2 실행시 import twisted 가 안되는 현상.
