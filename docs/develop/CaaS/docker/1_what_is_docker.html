Overview

Docker는 세계 최고의 소프트웨어 컨테이너 플랫폼입니다.

개발자는 동료와 코드를 공동 작업 할 때 Docker를 사용하여 "내 컴퓨터에서 작동하는"문제를 제거합니다.

운영자는 Docker를 사용하여 격리 된 컨테이너에서 나란히 실행하고 관리하여 컴퓨팅 밀도를 향상시킵니다.

기업은 Docker를 사용하여 Linux, Windows Server 및 Linux on Mainframe 응용 프로그램에 대해

새 기능을보다 빠르고 안전하고 안전하게 배포 할 수있는 민첩한 소프트웨어 제공 파이프 라인을 구축합니다.

What is a Container?

컨테이너는 공유 운영 체제에서 격리 된 형식으로 소프트웨어를 패키지화하는 방법입니다.

VM과 달리 컨테이너는 전체 운영 체제를 번들로 제공하지 않으므로 소프트웨어 작업에 필요한

라이브러리 및 설정만 필요합니다.

이는 효율적이고 가볍고 독립적인 시스템을 구축하고 배포 위치에 상관없이 소프트웨어가 항상 동일하게

실행되도록 보장합니다.

Docker For Developers

Docker는 개발 환경을 설정하고 구성하는 반복적인 작업을 자동화하여 개발자가 중요한 소프트웨어에

집중할 수 있도록합니다.

Docker를 사용하는 개발자는 복잡한 데이터베이스를 설치 및 구성 할 필요가 없으며 호환되지 않는

언어 툴 체인 버전을 전환하는 것에 대해 걱정할 필요가 없습니다.

앱이 도킹 (dockerized) 될 때, 그 복잡성은 쉽게 만들어지고, 공유되고 실행되는 컨테이너로 푸시됩니다.

새로운 코드베이스에 동료를 온 보드하는 것은 더 이상 소프트웨어를 설치하고 설정 절차를 설명하는 데

시간을 소요하지 않습니다.

Dockerfiles와 함께 제공되는 코드는 작업하기가 더 간단합니다.

종속성은 깔끔하게 패키지화 된 Docker 이미지로 가져오고 Docker 및 편집기가 설치된 모든 사용자는

몇 분 안에 응용 프로그램을 빌드하고 디버깅 할 수 있습니다.

Get Docker for Developers

ANY APP, LANGUAGE, OR STACK

비 호환성 또는 버전 충돌의 위험없이 모든 프로그래밍 언어로 작성된 Linux 및 Windows Server

컨테이너 응용 프로그램을 빌드, 테스트, 디버그 및 배포하십시오.

AWESOME DEVELOPER EXPERIENCE

온 보딩 시간을 65% 단축: 복잡한 멀티 컨테이너 애플리케이션을 신속하게 제작, 테스트 및 실행할 수

있으며 서버 및 개발자 머신에 소프트웨어를 설치하고 유지 관리하는 데 시간을 낭비하지 않아도 됩니다.

모든 종속성은 컨테이너에서 실행되어 "내 컴퓨터에서 작동하는"문제를 제거합니다.

BUILT-IN CONTAINER ORCHESTRATION

Docker에는 쉽게 구성 할 수있는 내장 된 클러스터 클러스터링이 함께 제공됩니다.

최소한의 설정만으로 제품을 모방 한 환경에서 앱을 테스트하고 디버그하십시오.

Docker for Ops

Docker는 소프트웨어 배포를 간소화합니다.

로드 블록없이 버그 수정 및 새로운 기능을 개발 및 배포하십시오.

실시간으로 응용 프로그램을 확장하십시오.

Docker는 어디서나 개발자와 IT 운영 팀의 비밀스런 무기이기 때문에 놀랍지 않게 자동으로,

안전하고 이식 가능한 앱을 제작, 배송, 테스트 및 배포 할 수 있습니다.

더 이상 위키, README, 긴 런북 문서 및 부실 정보가 담긴 포스트잇 노트가 없습니다.

Docker를 사용하는 팀은 개발, 준비 및 제작 과정에서 이미지가 동일하게 작동 함을 알고 있습니다.

새로운 기능과 수정 사항은 혼란, 놀라움 또는 중단 시간없이 고객에게 신속하게 전달됩니다.
