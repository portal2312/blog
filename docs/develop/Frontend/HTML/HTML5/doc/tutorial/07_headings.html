HTML Headings

중요도: <h1> > <h2> > <h3> > ... > <h6>

Note: <h1> tag 추가시 자동으로 앞뒤 공백을 추가한다

Headings Are Import

검색 엔진은 표제를 사용하여 웹 페이지의 구조와 내용을 색인화합니다.

사용자는 귀하의 페이지를 제목으로 훑어 봅니다.

문서 구조를 표시하려면 표제를 사용하는 것이 중요합니다.

중요도 순서로 표기합니다. (<h1> > <h2> > <h3> > ... <h6>)

Note:

heading 에만 HTML heading 를 사용하십시오. 제목을 사용하여 텍스트를 크게 또는 굵게 표시하지 마십시오.

HTML Horizontal Rules

tag: <hr>

정의: HTML 페이지에서 주제별 구분을 정의하며, 가장 자주 가로 규칙으로 표시됩니다.

용도: 내용을 분리, 변경을 정의한다.

The HTML <head> Element

태그: <head>

정의:

  • <head> 요소는 메타 데이터의 컨테이너입니다. (Heading 과 관련 없음)

  • HTML 메타 데이터는 HTML 문서에 대한 데이터입니다. 메타 데이터는 표시되지 않습니다.

방법: <html> 과 <body> 사이에 위치한다.

Note:

메타 데이터는 일반적으로 문서 제목, 문자 세트, 스타일, 링크, 스크립트 및 기타 메타 정보를 정의합니다.
