HTML Paragraphs

태그: <p> </p>

정의: 구문을 정의한다.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Note: 브라우저는 자동으로 단락 앞뒤에 공백을 추가한다

HTML Display

HTML이 어떻게 표시 될지 확신 할 수 없습니다.

크고 작은 화면과 크기가 조정 된 창은 다른 결과를 만듭니다.

HTML을 사용하면 HTML 코드에 여분의 공백이나 여분의 줄을 추가하여 출력을 변경할 수 없습니다.

브라우저가 페이지가 표시 될 때 여분의 공백과 여분의 행을 제거합니다.

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph
contains         a lot of spaces
in the source         code,
but the        browser
ignores it.
</p>

Don't Forget the End Tag

End tag 를 잊고 사용해도 브라우저에서 동적으로 지원하여 볼수 있다:

<p>This is a paragraph.
<p>This is another paragraph.

위의 예제는 대부분의 브라우저에서 작동하지만, 의존하지는 않습니다.

Note: 예기치 않은 결과를 얻을 수도 있습니다.

HTML Line Breaks

태그: <br>

정의:

  • 새 단락을 시작하지 않고 줄 바꿈을 원할 경우 <br> 사용하기.

  • empty tag 이며 끝 태그가 없을을 의미한다.

The Poem Problem

<p> 태그는 한 줄에 표시됩니다.

<p>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</p>

The HTML <pre> Element

서식이 미리 지정된 텍스트를 정의합니다.

<pre> 요소 안의 텍스트는 고정 너비 글꼴 (일반적으로 Courier)로 표시되며, 공백과 줄 바꿈을 모두 유지합니다

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>

Note: 입력한 형태 그대로 표시

예로 markdown 의 ````와 같다. 자주 `<code>` 태그와 함께 사용된다.
