Images Syntax

tag: <img>

<img> 태그는 비어 있으며 속성만 포함하고 닫는 태그가 없습니다.

src 속성은 이미지의 URL(웹 주소)을 지정합니다:

<img src="url">

The alt Attribute

alt 속성은 이미지에 대한 대체 텍스트를 제공합니다.

사용자가 어떤 이유로 든 볼 수없는 경우 (연결 속도가 느리거나 src 속성에 오류가 있거나 사용자가 화면 판독기를 사용하기 때문) 브라우저가 이미지를 찾을 수 없으면 alt 속성의 값이 표시됩니다.

<img src="img_chania.jpg" alt="Flowers in Chania">

Note: alt 속성은 필수입니다. 웹 페이지가 없으면 웹 페이지가 올바르게 검증되지 않습니다.

Image Size - Width and Height

style 속성을 사용하여 이미지의 너비와 높이를 지정할 수 있습니다.

값은 픽셀 단위로 지정됩니다 (값 뒤에 px 사용).

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

Note:
항상 이미지의 너비와 높이를 지정하기.
너비와 높이를 지정하지 않으면 이미지가 로드되는 동안 페이지가 깜박입니다.

Width and Height, or Style?

width, height 및 style 속성은 HTML5에서 유효합니다.

그러나 style 속성을 사용하는 것이 좋습니다.

내부 또는 외부 스타일 시트가 이미지의 원래 크기를 변경하는 것을 방지합니다:

<!DOCTYPE html>
<html>
<head>
<style>
img {
    width:100%;
}
</style>
</head>
<body>

<p>Using the style attribute:</p>
<!-- 이미지가 정사각형으로 표현됨-->
<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

<p>Using the width and height attributes:</p>
<!-- 이미지가 가로가 늘어져 표현됨 -->
<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

</body>
</html>

Images in Another Folder

지정하지 않으면 브라우저는 웹 페이지와 동일한 폴더에서 이미지를 찾으려고합니다.

그러나 이미지를 하위 폴더에 저장하는 것이 일반적입니다.

그런 다음 src 속성에 폴더 이름을 포함시켜야합니다:

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Images on Another Server

일부 웹 사이트는 이미지를 이미지 서버에 저장합니다.

사실, 세계의 모든 웹 주소에서 이미지에 액세스 할 수 있습니다:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Note: HTML File Paths

Animated Images

GIF 표준은 움직이는 이미지 허용한다:

<img src="/images/xxx.gif" alt="test" style="width:60px;height:60px;">

Using an Image as a Link

이미지를 링크로 사용하려면 <a> 태그 안에 <img> 태그를 중첩하면됩니다.

<a href="default.asp">
  <img src="smiley.gif" alt="smiley" style="width:42px;height:42px;border:0;">
</a>

Image Floating

CSS float 속성을 사용하여 이미지가 텍스트의 오른쪽이나 왼쪽으로 떠있게 됩니다:

<p>
  <img src="smiley.gif" alt="Smiley" style="float:right;width:60px;height:60px;">
  The image will float to the right of the text.
</p>

Image Maps

태그: <map>

정의:

  • image-map 은 click 가능한 영역이있는 이미지입니다.

  • name 속성은 <img> 의 usemap 속성과 연관되며 <img>와 <map> 사이의 관계를 만듭니다.

  • image-map 에서 click 가능한 영역을 정의하는 여러 <area> 태그를 포함합니다.

<img src="planets.gif" alt="Planets" usemap="#planetmap" style="width:145px;height:126px;">

<map name="planetmap">
  <area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm">
  <area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm">
  <area shape="circle" coords="124,58,8" alt="Venus" href="venus.htm">
</map>

Background Image

HTML 요소에 배경 이미지를 추가하려면 CSS 속성 background-image 를 사용합니다.

웹 페이지에 배경 이미지를 추가하려면 <body> 요소에 background-image 속성을 지정하십시오:

<body style="background-image:url('clouds.jpg')">

<h2>Background Image</h2>

</body>

단락에 배경 이미지를 추가하려면 P 요소에 background-image 속성을 지정합니다:

<body>

<p style="background-image:url('clouds.jpg')">
...
</p>

</body>

자세한 내용은 CSS Background Tutorial 에 참조하기.

The <picture> Element

HTML5는 이미지 리소스를 지정할 때 유연성을 더하기 위해 <picture> 요소를 도입했습니다.

<picture> 요소는 서로 다른 이미지 소스를 참조하는 여러 <source> 요소를 포함합니다.

이렇게하면 브라우저가 현재보기 및 / 또는 장치에 가장 잘 맞는 이미지를 선택할 수 있습니다.

각 <source> 요소에는 이미지가 가장 적합한시기를 설명하는 속성이 있습니다.

브라우저는 속성 값이 일치하는 첫 번째 <source> 요소를 사용하고 다음 <source> 요소는 무시합니다.

브라우저 창 (viewport)이 650px 이상인 경우 하나의 그림을 표시하고 그렇지 않은 경우 465px 보다 큰 이미지를 표시합니다:

<picture>
<source media="(min-width: 650px)" srcset="img_pink_flowers.jpg">
<source media="(min-width: 465px)" srcset="img_white_flower.jpg">
<img src="img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
</picture>

Note: 항상 <img> 요소를 <picture> 요소의 마지막 자식 요소로 지정하십시오. <img> 요소는 <picture> 요소를 지원하지 않는 브라우저 또는 <source> 태그가 일치하지 않는 브라우저에서 사용됩니다.

HTML Screen Reader

화면 판독기는 HTML 코드를 읽고 텍스트를 변환하며 사용자가 내용을 "들을"수있는 소프트웨어 프로그램입니다. 스크린 리더는 시각 장애인, 시각 장애인 또는 학습 장애인에게 유용합니다.

Chapter Summary

  • Use the HTML <img> element to define an image
  • Use the HTML src attribute to define the URL of the image
  • Use the HTML alt attribute to define an alternate text for an image, if it cannot be displayed
  • Use the HTML width and height attributes to define the size of the image
  • Use the CSS width and height properties to define the size of the image (alternatively)
  • Use the CSS float property to let the image float
  • Use the HTML <map> element to define an image-map
  • Use the HTML <area> element to define the clickable areas in the image-map
  • Use the HTML <img>'s element usemap attribute to point to an image-map

Note: 이미지로드에는 시간이 걸립니다. 큰 이미지는 페이지 속도를 늦출 수 있습니다. 이미지를 신중하게 사용하십시오.

Image Tags

Tag Description
<img> Defines an image
<map> Defines an image-map
<area> Defines a clickable area inside an image-map
<picture> Defines a container for multiple image resources
