Iframe Syntax

<iframe src="URL"></iframe>

Iframe - Set Height and Width

height 와 width 속성을 크기를 지정할 수 있다.

기본 크기 단위는 px이며, 백분율(%)도 가능하다.

<iframe src="URL" width="100%" height="300"></iframe>

Iframe - Remove the Border

기본적으로, iframe 은 border 가 존재한다.

border 삭제하려면, style 속성을 추가하고 CSS border 요소를 사용한다:

<iframe src="URL" width="100%" height="300" style="border:none;"></iframe>

반대로, border 를 디자인하려면:

<iframe src="URL" style="border:2px solid grey;"></iframe>

Iframe - Target for a Link

iframe은 링크 대상 프레임으로 사용할 수 있습니다.

링크의 대상 속성은 iframe의 name 속성을 참조해야 합니다.

<iframe height="300px" width="100%" src="demo_iframe.htm" name="iframe_a"></iframe>

<p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p>

HTML iframe Tag

Tag Description
<iframe> Defines an inline frame

Reference

https://www.w3schools.com/html/html_iframe.asp
