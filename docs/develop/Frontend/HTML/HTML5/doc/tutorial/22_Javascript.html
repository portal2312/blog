JavaScript는 HTML 페이지를 더욱 역동적이고 대화식으로 만듭니다.

The HTML <script> Tag

<script> 태그는 client 측 script(JavaScript)를 정의하는 데 사용됩니다.

<script> element는 script문을 포함하거나 src 속성을 통해 외부 script 파일을 가리킵니다.

자바 스크립트의 일반적인 용도는 이미지 조작, 양식 유효성 검사 및 콘텐츠의 동적 변경입니다.

HTML 요소를 선택하기 위해 JavaScript는 매우 자주 document.getElementById(id) 메소드를 사용합니다.

이 JavaScript 예제는 "Hello JavaScript!"라고 씁니다. id = "demo"인 HTML 요소에 추가합니다:

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

Tip: https://www.w3schools.com/js/default.asp

A Taste of Javascript

JavaScript 는 다음과 같이 할 수 있습니다.

HTML content 변경하기:

document.getElementById("demo").innerHTML = "Hello JavaScript!";

HTML style 변경하기:

document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";

HTML attribute 변경하기:

document.getElementById("image").src = "picture.gif";

The HTML <noscript> Tag

<noscript> 태그는 브라우저에서 스크립트를 비활성화했거나 클라이언트 측 스크립트를 지원하지 않는 브라우저를 가진 사용자를 위해 대체 컨텐츠를 제공하는 데 사용됩니다.

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

<noscript>Sorry, your browser does not support JavaScript!</noscript>

HTML Script Tags

Tag Description
<script> Defines a client-side script
<noscript> Defines an alternate content for users that do not support client-side scripts

Reference

https://www.w3schools.com/html/html_scripts.asp
