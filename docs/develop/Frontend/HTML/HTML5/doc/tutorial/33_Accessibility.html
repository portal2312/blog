HTML Accessibility

접근성을 염두에두고 HTML을 작성하십시오.

사용자가 사이트를 탐색하고 상호작용할 수있는 좋은 방법을 제공합니다.

HTML 코드를 가능한 의미론적으로 만드십시오.

그러면 코드가 방문자와 스크린 리더에게 이해하기 쉽습니다.

Semantic HTML

의미론적 HTML은 가능한 한 올바른 목적으로 올바른 HTML 요소를 사용한다는 것을 의미합니다.

의미론적 요소는 의미있는 요소입니다.

버튼이 필요한 경우 <button> 요소를 사용하십시오 (<div> 태그는 사용하지 마십시오).

Semantic:

<button>Click Me</button>

Non-semantic:

<div>Click Me</div>

의미론적 HTML은 웹 페이지의 내용을 큰 소리로 읽는 스크린 리더에게 문맥을 제공합니다.

버튼 예제를 염두에 두고:

  • 버튼에는 기본적으로 더 적합한 스타일이 있습니다.
  • 화면 판독기가 이를 단추로 식별합니다.
  • 초첨을 맞출수 있는
  • 클릭 가능한

키보드 전용 탐색 기능을 사용하는 사람들도 버튼에 액세스 할 수 있습니다.

마우스와 키를 사용하여 클릭 할 수 있으며 키보드의 탭 키를 사용하여 사이에 탭을 지정할 수 있습니다.

비의미론적 요소의 예: <div>와 <span> - 내용에 관해 아무것도 말하지 않습니다.

의미론적 요소의 예: <form>, <table> 및 <article> - 내용을 명확하게 정의합니다.

Headings Are Important

검색 엔진은 표제를 사용하여 웹 페이지의 구조와 내용을 색인화합니다.

사용자는 제목을 기준으로 페이지를 훑어 봅니다.

문서 구조와 다른 섹션 간의 관계를 표시하려면 머리글을 사용하는 것이 중요합니다.

<h1> 표제는 주 표제 다음에 <h2> 표제 다음 덜 중요한 <h3> 등으로 사용되어야합니다.

Note: 제목에만 HTML 표제를 사용하십시오. 제목을 사용하여 BIG or bold 텍스트를 작성하지 마십시오.

Alternative Text

어떤 이유로 든 사용자가 그것을 볼 수없는 경우 (연결 속도가 느리거나 src 속성의 오류 또는 사용자가 화면 판독기를 사용하기 때문에) alt 속성은 이미지의 대체 텍스트를 제공합니다.

alt 속성의 값은 이미지를 설명해야합니다:

<img src="img_chania.jpg" alt="Flowers in Chania">

browser 가 이미지를 찾을 수 없으면 alt 속성의 값이 표시됩니다:

<img src="img_chania.jpg" alt="Flowers in Chania">

Declare the Language

언어를 선언하는 것은 화면 판독기 및 검색 엔진에서 중요하며 lang 속성으로 선언됩니다.

영어로 웹 페이지를 표시하려면 다음을 사용하십시오:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

Use Clear Language

이해하기 쉬운 명확한 언어를 사용하고 화면 판독기에서 명확하게 읽을 수 없는 문자는 피하십시오:

  • 가능한 한 짧은 문장을 유지하십시오.
  • 대시를 피하십시오. 1-3 을 쓰는 대신 1 to 3을 사용.
  • 약어를 피하십시오. Feb 을 작성하는 대신 February을 작성하십시오.
  • 속어를 사용하지 마십시오.

Write Good Links

링크는 독자가 해당 링크를 클릭하여 얻을 수있는 정보를 명확하게 설명해야합니다.

좋은 링크와 나쁜 링크의 예:

<!DOCTYPE html>
<html>
<body>

<h2>Write Good Links</h2>
<p>A link should explain clearly what information the reader will get by clicking on that link.</p>

<p>Examples of bad links:</p>
<ul>
  <li><a href="#">Click here</a></li>
  <li><a href="#">Read more..</a></li>
  <li>Buy tickets to Mars <a href="#">here</a></li>
</ul>

<p>Examples of good links:</p>
<ul>
  <li><a href="#">Find out more about the HTML language</a></li>
  <li>Read more about <a href="#">how to eat healthy</a></li>
  <li><a href="#">Buy tickets to Mars here</a></li>
</ul>

<p>Note that these are just examples, and not clickable.</p>

</body>
</html>

Result:

Examples of bad links:

  • Click here
  • Read more..
  • Buy tickets to Mars here

Examples of good links:

  • Find out more about the HTML language
  • Read more about how to eat healthy
  • Buy tickets to Mars here
