Default Values

일반적으로 변수에 값을 할당 할 때 해당 변수에 이미 값이 있으면 이전 값을 덮어 씁니다.
그러나 Sass 라이브러리를 작성하는 경우 CSS 를 생성하기 위해 라이브러리 변수를 사용하기 전에 사용자가 라이브러리 변수를 구성하도록 허용 할 수 있습니다.

이를 가능하게하기 위해 Sass는 !default flag 를 제공합니다.
변수가 정의되지 않았거나 값이 null 인 경우에만 변수에 값을 지정합니다.
그렇지 않으면 기존 값이 사용됩니다.

!default 와 함께 정의된 변수는 @use rule 함께 모듈을 로딩중일 때 구성될 수 있도록 할 수 있습니다.
Sass 라이브러리는 종종 !default 변수를 사용자가 라이브러리의 CSS를 구성할 수 있도록 사용합니다.

구성 모듈을 로드하려면, 쓰십시오 @use <url> with (<variable>: <value>, <variable>: <value>).
구성된 값은 변수의 기본값보다 우선합니다. !default flag 가 있는 스타일 시트의 최상위 레벨에 작성된 변수 만 구성할 수 있습니다.

SCSS:

// _library.scss
$black: #000 !default;
$border-radius: 0.25rem !default;
$box-shadow: 0 0.5rem 1rem rgba($black, 0.15) !default;

code {
  border-radius: $border-radius;
  box-shadow: $box-shadow;
}

// style.scss
@use 'library' with (
  $black: #222,
  $border-radius: 0.1rem
);

=> CSS:

code {
  border-radius: 0.1rem;
  box-shadow: 0 0.5rem 1rem rgba(#222, 0.15);
}
