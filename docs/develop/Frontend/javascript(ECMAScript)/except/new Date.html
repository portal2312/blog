chrome 과 explore 에서 new Date({String}) 결과 값이 다름

YYYY-mm-dd HH:mm:ss 의 format 형식을 explore 는 지원 안함.

explore UTC 시간차(ISO-8601) 값도 표현해야 함.

시간 값 맞는지 확인할 것.

new Date('2017-07-01 00:00:00');
// Sat Jul 01 2017 00:00:00 GMT+0900 (KST) -- chrome
// Invalid Date  -- explorer


new Date('2017-07-01T00:00:00');
// Sat Jul 01 2017 00:00:00 GMT+0900 (KST) -- chrome
// Sat Jul 01 2017 00:00:00 GMT+0900 (KST) -- explorer
