Installation

여러 방법중 공식 페이지에서 소스 받아 해당 javascript Directory 에 추가하는 방법을 선택.

  1. Downloading jQuery > jQuery

  • Download the compressed, production jQuery x.x.x

    jquery-x.x.x.js

  • Download the uncompressed, development jQuery x.x.x

    jquery-x.x.x.min.js

  1. 소스 경로에 추가해서 html 에서 불러오기만 하면 끝.
