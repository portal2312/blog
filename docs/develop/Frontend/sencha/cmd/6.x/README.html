Manjaro (ArchLinux)

압축 파일을 다운로드해서 해제후

$ tar xzvfp SenchaCmd-6.2.1.29-linux-amd64.tar.gz
$ bash SenchaCmd-6.2.1.29-linux-amd64.sh

실행시 UI에서 제어해서 설치 단, 경로에 주의 할 것

/usr/local/lib 에 설치 후 실행 파일만 /usr/local/bin 에 등록

ln -s /설치경로/Sencha/Cmd/sencha /usr/local/bin/sencha

Upgrade

$ sencha upgrade
