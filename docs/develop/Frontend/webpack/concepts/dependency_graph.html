webpack - Dependency Graph

Table of contents

  • Further Reading
  • References

한 파일이 다른 파일에 의존 할 때마다, 웹팩은 이것을 파일을 의존성으로 취급합니다. 이를 통해 웹팩은 이미지 또는 웹 글꼴과 같은 비 코드 자산을 가져와 애플리케이션에 대한 종속성으로 제공 할 수 있습니다 .

webpack이 응용 프로그램을 처리 할 때 명령 줄 또는 구성 파일에 정의 된 모듈 목록에서 시작합니다. 이러한 진입 점 에서 시작하여 webpack은 애플리케이션에 필요한 모든 모듈을 포함 하는 종속성 그래프 를 재귀 적으로 빌드 한 다음 모든 모듈을 브라우저에서 load 할 수 있는 적은 수의 bundle(종종 단 하나 의 bundle)로 묶습니다.

애플리케이션 bundling 은 브라우저가 새 요청을 시작하는 동안 앱이 대기해야하는 횟수를 최소화하므로 HTTP/1.1 client 에 특히 강력합니다. HTTP/2 에서, 당신은 또한 사용할 수있는 코드 분할을 최상의 결과를 얻을 수 있습니다.
