Iterator (반복자)

  • 반복 구문에서 효과적으로 사용을 목적으로 함.

  • 어휘

    • iterator == 공간
    • iteration == 순차적 가져오는 행위 .next()
    • iterable == iteration 가능한

  • 사용 용도

    • Lazy evaluation
    • memory
    • 무한 표현
  iters = iter(list())  # 생성

  val = iters.next()  # 하나씩 반환

  # 반환할 내용이 없다면 요류 발생, iters를 index를 하나씩 소모
  for r in iters:
      print r

  # django
  for r in User.objects.all():  # 모든 사용자를 가져와서 memory 에 등록
      print r

  for r in User.objects.iterator():  # 적당한 단위로 끊어서 데이터를 가져옴
      print r

  print [x for x in xrange(1, 1000)]

  print (x for x in xrange(1, 1000))

  list()  # 유한한
  iter()  # 무한한

  list != iter()  # list는 iterable, iteration 가능, 단 iterator 는 아니다.

Generator (생성자)

  • iterator 처럼 반복 구문에서 사용.

  • for .. yield 로 구성

  • yield 를 이용해 값을 반환해도 종료되지 않고 상태를 유지함

  for return  # 종료
  for yield   # 종료되지 않고 유지하며 수행
