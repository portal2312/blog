Server Config

LDAP 서버가 기본 포트에서 로컬로 실행되지 않는 경우 서버를 가리 키도록 AUTH_LDAP_SERVER_URI를 설정하여 시작하려는 것입니다. 이 설정의 값은 LDAP 라이브러리가 지원하는 것이 될 수 있습니다. 예를 들어 openldap 을 사용하면 쉼표 또는 공백으로 구분 된 URI 목록을 차례대로 입력 할 수 있습니다.

AUTH_LDAP_SERVER_URI = "ldap://ldap.example.com"

서버 위치가 이보다 훨씬 동적 인 경우 URI를 반환하는 함수 (또는 호출 가능한 객체)를 제공 할 수 있습니다.

호출 가능 객체에는 하나의 위치 인수인 request가 전달됩니다.

이 요청은 모든 요청에서 호출되므로 비싼 작업 인 경우 일부 캐싱이 순서대로 수행됩니다.

from my_module import find_my_ldap_server

AUTH_LDAP_SERVER_URI = find_my_ldap_server
