해결:

settings.py 에 STATIC_ROOT 추가하기:

STATIC_ROOT = os.path.join(BASE_DIR, 'allstaticfiles')

이제 다시 명령을 실행하면 모든 정적 파일이 프로젝트 루트의 allstaticfiles 폴더에 수집됩니다.

다시 말하지만, STATIC_ROOT에 정적 파일을 PUT하지 마십시오.

이것이 바로 collectstatic을 실행 한 후에 파일이 자동으로 수집되는 위치입니다.

이제 프로덕션 환경에서 웹 서버가 해당 위치의 정적 파일을 제공하도록 구성 할 수 있습니다.

Nginx:

location /static/ {
    root   /my/project/allstaticfiles/;
}
