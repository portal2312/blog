Language: en

Documentation version: 1.11

django.contrib.staticfiles 는 프로덕션 환경에서 쉽게 제공 할 수있는 단일 위치에 각 응용 프로그램 (및 지정한 다른 모든 위치)에서 정적 파일을 수집합니다.

NOTE: See also

정적 파일 앱 및 사용 예제에 대한 소개는 [정적 파일 관리 (e.g. img, javascript, CSS)][]를 참조.
[정적 파일 배포][]에 대한 지침은 정적 파일 배포를 참조하십시오.

Settings

다음 설정에 대한 자세한 내용은 staticfiles 설정을 참조:

  • [STATIC_ROOT][]

  • [STATIC_URL][]

  • [STATICFILES_DIRS][]

  • [STATICFILES_STORAGE][]

  • [STATICFILES_FINDERS][]
