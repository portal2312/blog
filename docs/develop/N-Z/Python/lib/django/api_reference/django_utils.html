django.utils.decorators

method_decorator(decorator, name='')

  • 함수 데코레이터를 메서드 데코레이터로 변환합니다. 메서드나 클래스를 장식하는 데 사용할 수 있습니다. 후자의 경우 name 은 데코 레이팅 할 메서드의 이름이며 필수 속성입니다.

  • 데코레이터는 함수의 목록 또는 튜플 일 수도 있습니다. 호출 순서가 함수가 목록 / 튜플에 나타나는 순서가되도록 역순으로 줄 바꿈됩니다.

  • 클래스 기반 뷰 꾸미기(decorating class based views)

decorator_from_middleware(middleware_class)

decorator_from_middleware_with_args(middleware_class)
