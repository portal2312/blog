Django - add Apps

$ python manage.py startapp myext

add - models.py

# vi ./myproject/myext/models.py

# -*- coding:utf8 -*-
from django.db import models


class User(models.Model):
    userid = models.CharField(max_length=128)
    # ...

    def __str__(self):
        return userid
# ...

Table 생성. (models.py 기반 참조로 하여 생성)

  • makemigrations

    • SQL Query 생성 (./myproject/myext/migrations/0001_initial.py)
    • 직접 만들지는 않는다.

  • migrate : 생성.

$ python manage.py makemigrations myext
$ python manage.py migrate

# Operations to perform:
#   Apply all migrations: admin, auth, contenttypes, myext, sessions
# Running migrations:
#   Applying myext.0001_initial... OK

Apps - urls.py

# vi ./myproject/myext/urls.py

from django.conf.urls import url
from . import views


apps_name = 'myext'  # XXX: Add apps_name
urlpatterns = [
    url(r'^$', views.index, name='index'),
]

Projects - urls.py

# ./myprojects/urls.py

from django.conf.urls import include, url
from django.contrib import admin


urlpatterns = [
    url(r'^admin/', admin.site.urls),
    url(r'^myext/', include('myext.urls')),  # XXX: include
]
