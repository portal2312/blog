원인

  • 원인은 import 할 경로를 찾지 못한 문제.

  • 소스에서 현재 경로를 import 하면 잘 되는 것을 확인함.

    import os
import sys
sys.path.insert(0, os.path.abspath(os.path.curdir))
from echo import EchoFactory

해결

  • 사용할 디렉터리마다 init.py 추가해야 module 별 chain 식 import 할 수 있다.

  • 혹시 디텍처리나 파일명은 [-, ., 맨앞 숫자] 는 import 시 error 발생할 수 있다.

  • Python 환경 변수를 profile 등록해서 해결하기.

    vi ~/.bash_profile 또는 /etc/profile

PYTHONPATH=$HOME
export PYTHONPATH
