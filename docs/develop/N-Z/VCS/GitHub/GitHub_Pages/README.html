Jekyll

Installation

Jekyll on Ubuntu

sudo apt-get install ruby-full build-essential zlib1g-dev

root 사용자로 Ruby Gem을 설치하지 않는 것이 가장 좋습니다. 따라서 사용자 계정에 대한 gem 설치 디렉토리를 설정해야합니다. 다음 명령은 ~/.bashrc 파일에 환경 변수를 추가하여 gem 설치 경로를 구성합니다:

echo '# Install Ruby Gems to ~/gems' >> ~/.bashrc
echo 'export GEM_HOME="$HOME/gems"' >> ~/.bashrc
echo 'export PATH="$HOME/gems/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

Exception: 위 내용을 하지 않을 경우 오류 발생

gem install jekyll bundler

Fetching: safe_yaml-1.0.5.gem (100%)      
ERROR:  While executing gem ... (Gem::FilePermissionError)
    You don't have write permissions for the /var/lib/gems/2.5.0 directory.

install Jekyll:

gem install jekyll bundler

bundle init 실행하여 Gemfile 을 생성하기, 그리고 gem "jekyll" 추가하기:

bundle init
# Writing new Gemfile to ./Gemfile

echo 'gem "jekyll"' >> Gemfile
