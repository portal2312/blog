Settings

mirrorlist 추가

vi /etc/pacman.d/mirrorlist

상단에 추가

## South Korea

Server = http://ftp.kaist.ac.kr/ArchLinux/$repo/$os/$arch
Server = http://mirror.premi.st/archlinux/$repo/$os/$arch

pacman.conf

사용할 옵션에 대해 주석 해제

vi /etc/pacman.conf

Color
TotalDownload

Options

# get - package list in remote repository
pacman -Sy

# get - package list in remote repository
# refresh - system all package
# 설치된 패키지 업그레이드
pacman -Syu


# search
pacman -Ss <charactor>  # package in remote repository
pacman -Q <문자열>  # installed package


# install
pacman -S <package-name>  # package
pacman -U <package-name>  # install/refresh - files package


# delete
pacman -Scc  # cache
pacman -R <package-name>  # package


# view
pacman -Si <package-name>  # package info in remote repository
pacman -Ql <package-name>  # files in installed package
pacman -Qo <file-name>  # 지정된 파일이 포함된 package 출력

Upgrading packages

경고:

사용자는 시스템 유지 보수 #시스템 업그레이드 절의 지침에 따라 시스템을 정기적으로
업그레이드해야하며 맹목적으로 다음 명령을 실행하지 않아야 합니다.

아치는 전체 시스템 업그레이드만 지원합니다.
시스템 유지 보수 #부분 업그레이드는 지원되지 않으며 패키지 설치는 #세부 사항을 참조.

pacman은 단지 하나의 명령으로 시스템의 모든 패키지를 업데이트 할 수 있습니다. 이것은 시스템이 얼마나 최신인지에 따라 꽤 오래 걸릴 수 있습니다. 다음 명령은 리포지토리 데이터베이스를 동기화하고 구성된 리포지토리에없는 "로컬"패키지를 제외한 시스템 패키지를 업데이트합니다.

pacman -Syu

Reference

URL

  • https://wiki.archlinux.org/index.php/pacman

  • https://www.archlinux.org/pacman/pacman.conf.5.html
