/etc/profile

  • 시스템 전체 기능 및 별칭 환경 항목 등록.

  • '/etc/profile.d/'에 custom.sh 셸 스크립트를 만들어 환경에 맞게 사용자 지정 변경을 하면 나중에 업데이트 할 때 병합 할 필요가 없으므로 훨씬 좋습니다.

  • 전역 설정.

  • 로그인 시 load.

/etc/bashrc

  • 로그인 설정을 위한 시스템 전체 환경 및 시작 프로그램 함수 및 별칭 등록.

  • '/etc/profile.d/'에 custom.sh 셸 스크립트를 만들어 환경에 맞게 사용자 지정 변경을 하면 나중에 업데이트 할 때 병합 할 필요가 없으므로 훨씬 좋습니다.

  • 전역 설정.

  • bash shell 사용 load.

~/.bash_profile

  • 각 계정 별 설정 가능. (local-user control)

  • login 시 load.

  • 환경 변수와 bash가 수행될 때 실행되는 프로그램을 제어하는 지역적인 시스템 설정과 관련된 파일이다.

  • Login shell 로 사용.

  • Login 한 후에 한번만 실행.

~/.bashrc

  • 각 계정 별 설정 가능. (local-user control)

  • non-login, bash 실행시 load.

  • 별칭(alias), bash가 수행될 때 실행되는 함수를 제어, 지역적인 시스템 설정 관련된 파일.

  • 이 파일은 전역적인 설정 파일인 /etc/bashrc이 수행된 다음 바로 수행.

  • 사용자의 별칭과 함수, source 정의

~/.bash_logout

  • 각 계정 별 설정 가능. (local-user control)

  • logout 시 직전에 실행.

  • 프로그램에 관한 bash의 지역적인 시스템 설정과 관련된 파일.

BASH Commands

Commands Desc Example
export 환경 변수 등록 export PYENV_ROOT="/opt/pyenv"
source BASH Script File 실행 source '/opt/autoenv/activate.sh'
echo BASH 메시지 실행 echo 'export PYENV_ROOT="/opt/pyenv"' >> .bashrc
eval BASH 함수 실행 eval "$(pyenv init -)"
alias 별칭 등록 alias cdlogs="cd /opt/hts/logs"
bind 단축키 등록 bind -x '"\C-l": clear;'

실행 순서

/etc/profile
/etc/profile.d/a.sh
/etc/profile.d/b.sh
.bash_profile
.bashrc
/etc/bashrc

Example

  • [bind] Ctrl-l: clear > bind -x '"\C-l": clear;'k


echo '/etc/profile'
echo '' >> /etc/profile
echo '' >> /etc/profile
echo '## pyenv' >> /etc/profile
echo 'export PYENV_ROOT="/opt/pyenv"' >> /etc/profile
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> /etc/profile

echo '/etc/bashrc'
echo '' >> /etc/bashrc
echo '## pyenv' >> /etc/bashrc
echo 'eval "$(pyenv init -)"' >> /etc/bashrc
echo '## pyenv-virtualenv' >> /etc/bashrc
echo 'eval "$(pyenv virtualenv-init -)"' >> /etc/bashrc

echo '~/.bashrc'
echo '' >> ~/.bashrc
echo '## pyenv' >> ~/.bashrc
echo 'pyenv virtualenv system' >> ~/.bashrc
## autoenv 등록 (불편하니 사용하지 말자)
echo '## autoenv' >> ~/.bashrc
echo 'source /opt/autoenv/activate.sh' >> ~/.bashrc
