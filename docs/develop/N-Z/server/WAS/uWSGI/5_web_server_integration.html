Nginx

Nginx support

uWSGI 모듈은 공식 Nginx 배포판에 0.8.40 버전부터 포함되어 있습니다.

Nginx 0.7.x를 지원하는 버전은 uWSGI 패키지에서 유지 관리됩니다.

Unbit이 상업적으로 지원하는 안정적인 처리기입니다.
