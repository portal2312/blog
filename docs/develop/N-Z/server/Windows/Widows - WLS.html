Table of contents

Windows에서 linux 기능을 사용하기 원해서 시작하게 됨.

Windows - bash shell

Windows - ssh

Installation

  1. 설정 > 개발자 기능 사용
  • [x] 개발자 모드

재시작

  1. 제어판 > 프로그램 추가/삭제 > Windows 기능 켜기/끄기
  • [x] Linux 용 Windows 하위 시스템

재시작

  1. cmd.exe

prompt 에서 bash 입력시 ubuntu 설치 메시지 출력되는데 Y 로 설치.

Root passsword

해결:

$ sudo su

입력시 처음 생성한 계정의 passwd 입력하면 root로 계정변경됨.

이 때 변경하면 됨.

Programs

cmd.exe 를 권장.

Powershell 로 하면 error 나는 경우가 있음.
