< FTP 서버 구축 (IIS에서 FTP관련 서비스를 업데이트 한후 진행한다) >

1. iis 관리자 > 사이트 > 우클릭 > FTP 사이트 추가
   > 기본정보 입력 > SSL - 없음(체크) > 인증 - 기본("기본"만 체크) > 권한 부여 - 모든 사용자 > 사용 권한(읽기/쓰기 체크) > 마침


2. 계정생성 ( 서버관리자 > 구성 > 로컬 사용자 및 그룹 > 사용자 > 우클릭 > 새로운 사용자 )

- 사용자 이름 : 'test1' (Windows OS 계정과 관련없다)
- 암호: ****
- 다음 로그온 시 사용자가 반드시 암호를 변경해야 함(체크해제)
- 사용자가 암호를 변경할 수 없음(체크)
- 암호 사용 기간 제한없음(체크)

3. 생성된 FTP에 가상 디렉터리 추가

4. 가상 디렉터리 > 우클릭 > 사용자 권한 편집 > 보안 탭 > 편집 > 추가 > 고급 > 지금 찾기 > 'test1' 찾아서 추가 > 확인

5.  FTP 접속 테스트
