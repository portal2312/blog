[Window XP] 
1. [CMD > 창에서 실행] 
  a) sc create winexesvc binPath= C:\WINDOWS\WINEXESVC.EXE start= auto DisplayName= winexesvc 
  b) sc description winexesvc "Remote command provider for nOM monitoring" 
2. Window기본 방화벽해제 
3. 제어판 -> 관리도구 -> 로컬 보안 설정 
  a) 네트워크 보안 : Everyone 사용 권한을 익명 사용자에게 적용 -> Enable 
  b) > 네트워크 보안 : 로컬 계정에 대한 공유 및 보안 - > 일반 - 로컬 사용자를 그대로 인증 
4. 시작 -> 실행 -> DCOMCNF 실행 
  a) 콘솔 루트 -> 구성 요소 서비스 -> 컴퓨터 -> 내컴퓨터 우 측버튼 속성 클릭 
  b) 기본속성 탭 -> 이 컴퓨터에서 DCOM 사용 : 체크, 기본 인증 수준 : 연결 , 기본 가장 수정 : ID 
  c) COM 보안탭 -> 엑세스 권한 -> 제한값 편집 : "Anony mous", "Everyone" 추가 후 로컬 엑세스, 원격 엑세스 허용 체크 
  d) COM 보안탭 -> 엑세스 권한 -> 제한값 편집 : "Anonymous", "Everyone" 추가 후 로컬 엑세스, 원격 엑세스 허용 체크
