Keybindings

Commands Description
tmux new -s "SESSION_NAME" Create session
tmux ls View session list
tmux att -t Connect session
ctrl + a + d Close session
ctrl + a + c Create Tab
ctrl + d Close current panel
ctrl + A + v (= ctrl + a + %) Create vertical split panel
ctrl + A + s Create horizen split panel
ctrl + a + 방향키 Move split panel
ctrl + a + e On input key all panel
ctrl + a + E Off input key all panel
ctrl + a + space bar 현재 분할된 화면을 균등 분할로 바꾸기
ctrl + a + t 시간보기
ctrl + c 작업나오기
ctrl + l 화면 clear

view 모드로 전환: - Ctrl + a + [: view 모드로 전환 - Ctrl + c: 나오기 -> 기존모드(명령어모드)로 전환

세션 죽이기: tmua kill-session -t mkkim

tmux 세션중 복사, 붙여넣기:

  • 읽기모드: Ctrl + b + [
  • 선택: Ctrl + space
  • 복사: Ctrl + enter
  • 붙여넣기: Ctrl + b + ]

tmux.conf 

set -g status-left '#[fg=green] #(echo ${SSH_CONNECTION%%%% *}) > #[fg=yellow]#(ifconfig -a | grep "inet " | grep "Bcast:" | awk '"'{print $2}'"' | awk -F: '"'{print $2}'"') #[fg=green]> (#S) > #(whoami)@#h'

ex) '#fg=green #(whoami)@#h'

archlinux:

set -g status-left '#[fg=green] #(echo ${SSH_CONNECTION%%%% *}) > #[fg=yellow]#(ifconfig -a | grep "inet " | grep "netmask " | awk '"'{print $2}'"' | awk -F: '"'{print $1}'"' | head -1) #[fg=green]> (#S) > #(whoami)@#h'
