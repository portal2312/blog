python.vim

Download python.vim.

Move python.vim -> /etc/vim/syntax/python.vim.

Edit /etc/vim/vimrc:

syntax on
filetype indent plugin on
let python_highlight_all=1

pydiction

Download pydiction.

Move pydiction -> /etc/vim/plugins/pydiction.

Create structure:

mkdir -p /etc/vim/after/ftplugin
cp -rvp /etc/vim/plugins/pydiction/after/ftplugin/python_pydiction.vim /etc/vim/after/ftplugin/

Edit /etc/vim/vimrc:

filetype plugin on
let g:pydiction_location="/etc/vim/plugins/pydiction/complete-dict"
let g:pydiction_menu_height=3
