작업중인 파일을 임시로 저장해 둘수 있음.

작업중일 때 pull 해야되는 상황.

$ git status
$ git stash
$ git stash list

# 적용
$ git stash apply <>
$ git stash pop <>

# stash 원복
$ git status

Stash 되돌리기

$ git stash show -p stash@{0} | git apply -R

Stash 삭제하기

$ git stash drop stash@{0}

Reference

https://git-scm.com/book/ko/v2/Git-%EB%8F%84%EA%B5%AC-Stashing%EA%B3%BC-Cleaning
