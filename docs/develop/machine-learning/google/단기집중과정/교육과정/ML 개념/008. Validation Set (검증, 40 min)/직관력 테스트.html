이 모듈을 시작하기 전에 학습 및 테스트 세트에 제시된 학습 프로세스를 사용하는 데 문제점이 있는지 확인해 보시기 바랍니다.

테스트 세트 및 학습 세트를 사용하여 모델 개발 반복 과정을 진행하는 프로세스를 살펴보았습니다. 각 반복 과정에서는 학습 데이터를 학습하고 테스트 데이터로 평가하면서 테스트 데이터에 대한 평가 결과를 근거로 학습률, 특성 등의 다양한 모델 초매개변수를 선택하고 변화를 줍니다. 이 접근법에 잘못된 점이 있나요? 정답을 하나만 고르세요.

  • 이 절차를 많이 반복하면 특정 테스트 세트의 특이성에 은연중에 적응하게 됩니다.

정답입니다. 특정 테스트 세트로 자주 평가할수록 해당 테스트 세트에 대한 암시적 과적합이 나타날 위험이 커집니다. 다음 과정에서 더 나은 방식을 살펴보려고 합니다.

  • 문제가 없습니다. 학습 데이터로 학습하고, 별도로 분리해 둔 테스트 데이터로 평가합니다.

실제로는 미묘한 문제점이 있습니다. 이 방식으로 반복을 아주 많이 진행한다면 어떻게 될지 생각해 보세요.

  • 이 방식은 연산 효율이 낮습니다. 기본 초매개변수 세트를 선택하고 그대로 사용하여 리소스를 절약해야 합니다.

이러한 유형의 반복 과정은 많은 리소스를 사용하지만 모델 개발에서 핵심적인 부분을 차지합니다.
초매개변수 설정은 모델의 품질에 막대한 영향을 주므로 항상 충분한 시간과 연산 리소스를 투입하여 가능한 한 최고의 품질을 달성할 필요가 있습니다.
