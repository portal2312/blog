특성 교차로 돌아가기

특성 교차 는 좋은 방법이지만 희소 특성 교차 시 문제가 발생할 수 있다.

아무리 학습 데이터가 많더라도 이러한 조합은 매우 드물게 나타난다.

  • 주의: 희소 특성 교차는 특성 공간을 크게 늘릴 수 있습니다.
    • 예: 특성중 하나는 검색어에 포함된 단어고 다른 특성은 검색할 동영상 경우
  • 문제:
    • 모델 크기(RAM)가 매우 커질 수 있음
    • '노이즈' 계수(과적합의 원인)

과적합시 정규화 하기

  • L0 정규화
    • 노이즈가 심한 개수만 제거하기.
    • 가중치 0 설정하여 없애는 것.
    • 이 경우 0 이 아닌 가중치가 있으면 패널티 주기.
    • 볼록한 모양이 아니기 때문에 최적화가 어렵다. (후 행쉼표 문제)
