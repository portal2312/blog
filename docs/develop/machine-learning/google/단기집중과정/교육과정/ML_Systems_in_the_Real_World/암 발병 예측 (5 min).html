이 강의에서는 암 예측과 관련된 실제 ML 문제*를 디버그합니다.

학습 목표:

  • 실제의 ML 모델에서 결함을 파악합니다.

요약

암 예측 모델

  • 환자의 나이, 성별, 질병, 병원 이름, 생명 징후, 테스트 결과등 특징 추출
  • 테스트와 학습 데이터가 섞이지 않도록 주의 > 병원 이름: 병원에 아직 지정되지 않은 새로운 환자일 경우, 이 정보를 알 수 없다.

병원 전문 분야가 다른데 알려주므로 암 병원의 환자 정보는 대부분 암환자라 예측하게 되는 것.

라벨 유출:

  • 모델의 의사 역할시 판단할 수 있는 정보를 미묘한 방식으로 모델에 노출하게 된 것
  • 약간의 학습 라벨이 기능에 유출되어 모델의 속임수를 허용하는 것

Reference

https://developers.google.com/machine-learning/crash-course/cancer-prediction
