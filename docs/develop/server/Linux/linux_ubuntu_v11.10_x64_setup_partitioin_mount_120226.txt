< 우분투 설치 관련 파티션 나눌 때 마운트 설명>

마운트(mount) : 마운트는 오직 리눅스에서만 사용되는 개념,
리눅스의 불완전함을 보완해주기 위해 일종의 방어대책 같은 것(리눅스는 불완전한 OS).

리눅스는 하드디스크의 파티션과, CD/DVD 혹은 USB같은 것들을 사용해주기 위해서
리눅스의 특정한 위치에다가 연결을 해주어야만 사용 가능하게 되는데 이 때 연결해주는
과정을 바로 마운트라고 부릅니다.


/ : 루트 파티션. (권장: 1 GB 이상)

/boot : LILO 나 GRUB 같은 로더가 부팅 시 이 디렉토리에 있는 커널 이미지를 사용
	커널 이미지 및 system.map과 같은 파일이 위치함. (권장: 500 MB)

/usr : 사용자를 위한 공간, 일반적으로 응용 프로그램 설치 시 및 저장 사용됨.
	(권장 : 설치 프로그램에 비례)

/usr/local : 프로그램과 데이터 파일 저장됨. (권장: 사용자의 비례)

/tmp : 각종 작업에 사용되는 임시파일들을 저장. (권장: 2 GB)

/home : root 사용자를 제외한 사용자의 home 디렉터리가 위치. (권장: 사용자에 비례)

/opt : 상용 프로그램(=상업용 프로그램)이 위치함. (권장 : 없음)

/var : 로그, 캐시파일 등 저장됨. (권장: 2 GB)

/srv : 서비스 디덱토리, 주로 인터넷 관련 파일이 위치함. (권장 : 없음)

swap 파티션 : RAM이 부족할 때 사용되는 공간. ( 권장: RAM 크기에 x2배)
