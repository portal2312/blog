version: 17.0

arch linux 기반.

인기있는 linux top 3.

Installation

어렵지 않음.

Partition mount

free 할꺼냐 manual 로 할꺼냐.

free 해도 /usr 등 전부 root 쪽으로 등록됨.

free 로 설정.

Package Manager (pacman 사용)

참조: linux - pacman.md

한글 Keyboard 설정

$ pacman -Ss ibus-hangul  # search
$ pacman -S ibus-hangu  # install

vi ~/.bashrc

export GTK_IM_MODULE=ibus
export XMODIFIERS=@im=ibus
export QT_IM_MODULE=ibus

윈도우 키 > ibus 검색 후 실행 > alert 창 확인 > 확인 >

Ibus preferences 창 뜸 - Input Method(입력방식) - Add - korea 검색 후 추가 >

윈도우 키 > Session and Startup > Add > ibus/ibus/ibus-daemon 차례로 입력 >

재시작 (여기까지 해도 잘 됨)

윈도우 키 > ibus 검색 후 실행 > Input Method > preferences > 한글변경키 변경

Chrome

$ pacman -Ss chrome
$ pacman -S extra/chromium

실행시 Edit Application... 에서 아래와 같이 수정 후 실행

chromium --no-sandbox

Python

$ tar xvfp tar.xz

$ ./configure
$ make
$ make install
