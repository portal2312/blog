2. 라우터

네트웍 프로토콜로부터 얻은 정보를 기반으로 여러 다른 네트웍들과 직접적인 통신을 합니다.

인터넷 상에는 수백수천만대의 컴퓨터들이 서로 연결 되어있습니다.

각각의 컴퓨터들은 자신이 속한 네트웍에서 각자의 어드레스를 가지고 통신을 하고

자신의 네트웍에 속하지 않은 컴퓨터와 통신을 하기위해 라우터로부터 해당 컴퓨터의 네트웍 정보를 요청합니다.

라우터는 독립된 네트웍간의 연결을 가능하게하여 각 지역 네트웍의 구성원들이 원하는 컴퓨터와 통신을 할수 있게 해줍니다.
