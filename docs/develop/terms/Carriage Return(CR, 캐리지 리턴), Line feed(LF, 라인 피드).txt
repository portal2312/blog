# 캐리지 리턴(carriage return)

문자의 새 줄을 시작하는 데 쓰이는 제어 문자나 그 구조를 가리킨다.

컴퓨터 환경에서는 간단히 CR로 줄여 쓴다.

원래 캐리지 리턴은 타자기의 구조나 레버를 가리키는 용어였다.

최초의 전기식 캐리지 리턴은 1960년에 스미스 코로나가 전기 타자기에 추가하였다.

비영어권의 자판 개선을 위해 캐리지 리턴을 가리키는 ↵ (U+21B5) 기호가 도입되기도 했다.

아스키와 유니코드에서 캐리지 리턴은 13 (16진수 0D)으로 정의되며,

Ctrl+M (^M)을 통해서도 볼 수 있다.

이에 영향을 받은 C 프로그래밍 언어 및 기타 수많은 언어들에서 \r은 이 문자를 가리킨다.



- (본래 의미) 캐리지 리턴(CR)은 해당 라인에서 커서를 맨 앞으로 보내는 것이고

- 라인 피드(LF)는 다음 라인으로 이동하는 것입니다.

- 두 동작을 합치면 뉴라인 동작을 하게 되는데 이것은 과거의 느린 프린터에서 

  물리적인 속도차이를 커버하기 위해 만들어졌지만 현재에는 의미가 없습니다.

- Unix 계열 시스템에서는 LF로 뉴라인을 표시하고 Windows 는 CR+LF로 표현합니다.

- 그리고 정규식에서는 LF하나로 표현합니다.
