Installation

$ su -
$ yum install screen

screen dependencies 설치:

$ yum install  ncurses-devel texinfo pam-devel libtool

Commands & Keybindings

1. 생성
> screen -S mkkim

2. 스크린 나가기
Crtl + d

3. 스크린 리스트 보기
> screen -list / screen -ls

4. "mkkim" 세션접속하기
> screen -r  mkkim
> screen -dr mkkim (해당 세션의 로그인을 끈고 다시 접속)

5. 창 제어 명령어

> Ctrl + a + c   (새로운 창 생성)

> Ctrl + a + d   (현재 창 닫기)

> Ctrl + a + w   (현재 모든 창 번호 보기)

> Ctrl + a + [0~9] (해당 창으로 이동 = 0~9 창 번호선택)
> Ctrl + a + a     (이전 창으로 이동)

6. 스크인 세션 kill
> screen -X kill
