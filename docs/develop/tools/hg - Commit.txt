< 커밋하기는 순서 >
0. 개념

hg는 해당 지정된 폴더가 있어야 해당된 폴더가 있는 곳에서

사용할 수 있다(hg commit 등)



1. hg stat
= 전체 Commit된 Data와 이전 수정된 기록을 비교해서
  수정된 사항이 있는지 판별
  (M = 변경, A = 추가, 삭제 = R, 폴더상 사라진 것 = !, 관리대상이 아님 = ?)



2-1. hg glog -l(L) "숫자" = 마지막 Commit된 최근 기록과 메모를 볼 수 있음



2-2. hg vimdiff "파일명" = "파일명"을 기준으로 1:1로 비교
     이전 기록보기
     hg vimdiff -r 번호 "파일명"



3. hg commit = 마지막 서버에 Commit




4. hg revert "파일명" = 이전 "파일"로 되돌리기


5. hg 이력에서 이전소스 가져오기

> hg cat ./common.js -r 109 > ./common109.js
