ImportError: cannot import name _remove_dead_weakref

원인: virtualenv 로 인한 broken 현상

Traceback (most recent call last):
  File "/home/zen/lib/pycharm-2017.3.2/helpers/packaging_tool.py", line 203, in main
    retcode = do_untar(name)
  File "/home/zen/lib/pycharm-2017.3.2/helpers/packaging_tool.py", line 138, in do_untar
    import tarfile
  File "/usr/lib/python2.7/tarfile.py", line 52, in <module>
    import copy
  File "/usr/lib/python2.7/copy.py", line 52, in <module>
    import weakref
  File "/usr/lib/python2.7/weakref.py", line 14, in <module>
    from _weakref import (
ImportError: cannot import name _remove_dead_weakref

해결:

  1. project interpreter 에서 새로운 가상환경 만들기(설정하기).

  2. 이전 가상환경의 site-packages 를 새로운 가상환경에 폴더 전체를 옴기기

  3. 간혹 packages 중 build 나 compile 하여 동작하는 package 들은 에러 메시지를 확인하여 재설치를 하기 (psycopyg2 등).

  4. 누락된 파일이 있는지 확인하기.

Reference:

https://askubuntu.com/questions/981663/python2-7-broken-by-weakref-import-error-please-help

Inotify Watches Limit

echo 'fs.inotify.max_user_watches = 524288' >> /etc/sysctl.conf

sudo sysctl -p --system

https://confluence.jetbrains.com/display/IDEADEV/Inotify+Watches+Limit
