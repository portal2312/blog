Deployment

소스 수정시 즉시 동기화하기 위해서 등록한다.

Connection

  • Type: SFTP
  • SFTP host: {remote-server-ip}
  • Port: 22
  • Root path: {remote-servier-projects-path}
  • User name: {remote-server-connect-account}
  • Auth type: {passwwd} or {openSSH config and authentication agent}

Mapping

  • Local path: {projects-path}
  • Deployment path on server: /
  • Web Path on server: /

Note: Deployment path on server: / 인 이유

Connection > Root path 값이 {remote-servier-projects-path} 이기 때문이다.

Automatic

action(ctrl + shift + a) > Deployment: Automatic Upload On

Run/Debug Configurations

Django server 선택하기

Configuration

  • Host: 0.0.0.0
  • Port: 8000
  • Python interpreter: Remote Python x.x (ssh://**)

OK 하기

Note: Debugging 은 Browser URL 에 http://Remote-Server-IP:Port 입력하기.

Port 기준으로 Debugging 하기 때문에 break point 시 운영에는 문제없다.
