Keymap

Settings > Keymap

Plug-ins > IdeaVim

추가하기:

Shortcut Action
ctrl+alt+g Goto Declaration
ctrl+A+S Split window horizontally
ctrl+A+V Split window vertically

Main menu > Edit > Find

추가하기:

Shortcut Action
ctrl+A+F Find Next
CTRL+Z+C Collapse All

Other Settings

Sencha

all check

Vim Emulation

Plugins: IdeaVim plugin 설치 필수.

해당 단축키를 Vim -> IDE 로 변경하면 전역 설정처럼 동작함.

변경하기:

Shortcut IDE Action Handler
ctrl+c Copy Vim -> IDE

Shortcuts

유용한 단축키:

Keys Action Comments
ctrl+shift+n Go to file 파일 찾기
ctrl+shift+a Find Action 실행할 수 있는 기능 찾기
ctrl+shift+f Find in Path 문자열로 특정 파일의 문자 찾기
ctrl+tab switcher
ctrl+w+h Split Window move h, j, k, l vimmode 에서
alt+Left Select Previous Tab
alt+Right Select Next Tab
SHIFT + F10 Run 실행
SHIFT + F9 Debugging 실행
CTRL + F2 실행 중지
CTRL + F8 Break point line (add, remove)

Theme

Material Theme (Plugin)

메뉴 크기 조정하기:

  • /Settings/Appearance & Behavior/Material Theme/Compact/:
    • [x] Compact Menus

글자 크기 조정하기:

  • Editor/Color Scheme/Color Scheme Font:
    • Size: 14
    • Line spacing: 14
  • Editor/Color Scheme/Console Font:
    • Size: 12
    • Line spacing: 12

Project 글자 크기 조정하기:

  • /Settings/Appearance & Behavior/Material Theme/Project View/:
    • [x] Font Size: 12
